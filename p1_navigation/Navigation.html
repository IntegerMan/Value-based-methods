2. Examine the State and Action Spaces¶

The simulation contains a single agent that navigates a large environment. At each time step, it has four actions at its disposal:

  • 0 - walk forward
  • 1 - walk backward
  • 2 - turn left
  • 3 - turn right

The state space has 37 dimensions and contains the agent's velocity, along with ray-based perception of objects around agent's forward direction. A reward of +1 is provided for collecting a yellow banana, and a reward of -1 is provided for collecting a blue banana.

Once this cell is executed, you will watch the agent's performance, if it selects an action (uniformly) at random with each time step. A window should pop up that allows you to observe the agent, as it moves through the environment.

env_info = env.reset(train_mode=False)[brain_name] # reset the environment

while True:
    action = np.random.randint(action_size)        # select an action
    env_info = env.step(action)[brain_name]        # send the action to the environment
    next_state = env_info.vector_observations[0]   # get the next state
    reward = env_info.rewards[0]                   # get the reward
    done = env_info.local_done[0]                  # see if episode has finished
    score += reward                                # update the score
    state = next_state                             # roll over the state to next time step
    if done:                                       # exit loop if episode finished
        break

from dqn_agent import Agent
import torch
from collections import deque


def dqn(n_episodes=1000, max_t=1000, eps_start=1.0, eps_end=0.1, eps_decay=0.99, window_size=100, threshold=13):
    """Deep Q-Learning.
    
    Params
    ======
        n_episodes (int): maximum number of training episodes
        max_t (int): maximum number of timesteps per episode
        eps_start (float): starting value of epsilon, for epsilon-greedy action selection
        eps_end (float): minimum value of epsilon
        eps_decay (float): multiplicative factor (per episode) for decreasing epsilon
        window_size (int): the number of most recent scores to use when averaging performance
        threshold (float): when the algorithm should early exit the training process once a minimum window score is reached
    """
    scores = []                        
    scores_window = deque(maxlen=window_size)  
    eps = eps_start
    
    # Run all episodes (will early exit if threshold is reached)
    for i_episode in range(1, n_episodes+1):
        
        # reset the environment & get the current state
        env_info = env.reset(train_mode=True)[brain_name]
        state = env_info.vector_observations[0]
        score = 0
        
        # Run up until the max time we were given
        for t in range(max_t):
            
            # Get an action and cast it to prevent an odd bug with the ML agents package
            action = agent.act(state, eps)
            action = action.astype(np.int32)
            
            # Take the action and advance the simulation
            env_info = env.step(action)[brain_name]
            next_state = env_info.vector_observations[0]
            done = env_info.local_done[0] 
            
            # Learn from the scenario
            reward = env_info.rewards[0]
            agent.step(state, action, reward, next_state, done)
            
            # Update score and game state
            state = next_state
            score += reward

            # If the simulation reached its end, break out of time loop early
            if done:
                break 
        
        # Log the scores for active diagnostics and final charting later
        scores_window.append(score)
        scores.append(score)
        print('\rEpisode {}\tAverage Score: {:.2f}'.format(i_episode, np.mean(scores_window)), end="")
        if (np.mean(scores_window) >= threshold):
            break

        # Decrease epsilon
        eps = max(eps_end, eps_decay*eps)
        
        # Report partial progress
        if i_episode % 50 == 0:
            print('\rEpisode {}\tAverage Score: {:.2f}'.format(i_episode, np.mean(scores_window)))
    
    # save the model
    torch.save(agent.qnetwork_local.state_dict(), 'Model.pth')
    
    return scores
